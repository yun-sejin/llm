Airflow에서 DAG를 사용 중지하려면, DAG의 상태를 "paused"로 설정할 수 있습니다. 이를 통해 DAG가 스케줄에 따라 실행되지 않도록 할 수 있습니다. DAG를 사용 중지하는 방법에는 여러 가지가 있습니다:

1. Airflow UI를 사용하여 DAG 사용 중지
Airflow 웹 UI에서 DAG를 사용 중지할 수 있습니다.

Airflow 웹 UI에 접속합니다.
DAG 목록에서 사용 중지하려는 DAG를 찾습니다.
DAG 이름 옆에 있는 "Pause" 버튼을 클릭합니다.
2. Airflow CLI를 사용하여 DAG 사용 중지
Airflow CLI를 사용하여 DAG를 사용 중지할 수 있습니다.
airflow dags pause <dag_id>

예를 들어, dag_id가 "delete"인 DAG를 사용 중지하려면 다음 명령어를 실행합니다:

3. DAG 파일에서 사용 중지 설정
DAG 파일에서 is_paused_upon_creation 속성을 사용하여 DAG가 생성될 때 자동으로 사용 중지되도록 설정할 수 있습니다.
airflow dags pause delete



sample.py


import pendulum
import logging
from airflow.decorators import dag, task
from airflow.operators.empty import EmptyOperator
from airflow.models import Variable

logger = logging.getLogger(__name__)

dag_info = Variable.get("dag_info", deserialize_json=True)

@dag(
    dag_id="delete",
    schedule="*/10 * * * *",
    start_date=pendulum.parse(dag_info.get("start_date", "2024-04-10T00:00:00Z")).in_timezone("UTC"),
    catchup=False,
    is_paused_upon_creation=True,  # DAG가 생성될 때 자동으로 사용 중지
)
def delete():
    @task
    def check_upload_delete():
        logger.info("This is the first task.")
        # pass  # 실제 작업이 없으므로 pass 사용

dag_instance = delete()


설명
Airflow UI를 사용하여 DAG 사용 중지:

Airflow 웹 UI에서 DAG 목록에서 "Pause" 버튼을 클릭하여 DAG를 사용 중지할 수 있습니다.
Airflow CLI를 사용하여 DAG 사용 중지:

airflow dags pause <dag_id> 명령어를 사용하여 DAG를 사용 중지할 수 있습니다.
DAG 파일에서 사용 중지 설정:

is_paused_upon_creation=True 속성을 사용하여 DAG가 생성될 때 자동으로 사용 중지되도록 설정할 수 있습니다.
이 방법들을 통해 Airflow에서 DAG를 사용 중지할 수 있습니다. DAG가 사용 중지되면 스케줄에 따라 실행되지 않으며, 필요할 때 다시 활성화할 수 있습니다.
