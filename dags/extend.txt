와 extend는 Python의 리스트 메서드로, 리스트에 요소를 추가하는 데 사용됩니다. 그러나 두 메서드는 서로 다른 방식으로 요소를 추가합니다.

append
설명: append 메서드는 리스트의 끝에 단일 요소를 추가합니다.
사용법: list.append(element)
예제:
extend
설명: extend 메서드는 리스트의 끝에 다른 리스트의 모든 요소를 추가합니다.
사용법: list.extend(iterable)
예제:
차이점
append는 리스트에 단일 요소를 추가합니다. 추가되는 요소가 리스트일 경우, 리스트 자체가 하나의 요소로 추가됩니다.
extend는 리스트에 다른 리스트의 모든 요소를 추가합니다. 추가되는 리스트의 각 요소가 개별적으로 추가됩니다.
예제 비교
append 사용
extend 사용
이 예제에서 append는 리스트 [4, 5, 6]을 하나의 요소로 추가하는 반면, extend는 리스트 [4, 5, 6]의 각 요소를 개별적으로 추가합니다.


my_list = [1, 2, 3]
my_list.append(4)
print(my_list)  # 출력: [1, 2, 3, 4]
my_list = [1, 2, 3]
my_list.extend([4, 5, 6])
print(my_list)  # 출력: [1, 2, 3, 4, 5, 6]

my_list = [1, 2, 3]
my_list.append([4, 5, 6])
print(my_list)  # 출력: [1, 2, 3, [4, 5, 6]]

my_list = [1, 2, 3]
my_list.extend([4, 5, 6])
print(my_list)  # 출력: [1, 2, 3, 4, 5, 6]
